## Development

1. Clone this repository

   ```bash
   git clone https://github.com/Deri-Kurniawan/3d-portfolio.git
   ```

2. Install dependencies

   ```bash
    npm install
   ```

3. Set up environment variables

   Copy `example.env` to `.env` and fill your own environment variables if needed.

4. Run the development server

   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
6. Start editing the code in the `src` folder. especially in the `src/config/index.ts` file to update your personal information.
7. Happy coding!
